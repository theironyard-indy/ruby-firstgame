Free Crash Course: Build Your First Game

View on GitHub Download .zip Download .tar.gz

The Build Your First Game course will cover the Ruby programming language from the ground up to a functional command line game or two.

Along the way, you'll learn about:

  • Variables
  • Classes
  • Functions
  • Blocks
  • Data Structures
  • Handling user input

Of note: We won't be delving into Ruby on Rails, the popular Ruby-based web application framework. Ruby itself is fun enough to fill an evening.

Your presenter

Chris Vannoy is a Backend Engineering Instructor at The Iron Yard's Indianapolis campus. He's been developing in Ruby (and Rails) for nine years - since the early days of Rails.

What you'll need

We're going to be using Cloud9 for this workshop so we can all get up and running with a development environment quickly.

Go ahead and sign up for a free account, but we'll walk through setting up your development environment in Cloud9 in the workshop itself.

Macs, PC, Linux boxes are all welcome. Come as you are (computer-wise).
